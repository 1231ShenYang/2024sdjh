安装基础软件

>apt install git cmake g++ gcc clang libomp-dev -y

在该处安装libomp-dev是因为发现clangd在代码编辑时对omp.h会报错，这是由于GCC和Clang的openMP库并不相同(libgomp libomp)，因此使用libomp并在~/.config/clangd/config.yaml中加入对应omp.h的位置就可以解决报错

>wget https://www.doxygen.nl/files/doxygen-1.11.0.linux.bin.tar.gz \ >tar -xzf ./*.gz \ >cd doxygen* && cd bin && mv * /usr/local/bin \ >cd ../../ && rm -rf doxy* \ >apt直接安装的doxygen版本较低为1.9.8，最新为1.11.0
